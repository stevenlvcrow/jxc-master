# 物品模块字段对齐说明

## 1. item_profile（数据库）与 MyBatis Bean 对齐

表：`dev.item_profile`

| 数据库列 | MyBatis DO 字段 | 说明 |
|---|---|---|
| `scope_type` | `scopeType` | 作用域类型 |
| `scope_id` | `scopeId` | 作用域ID |
| `item_id` | `itemId` | 物品ID |
| `item_code` | `itemCode` | 物品编码 |
| `detail_json` | `detailJson` | 完整物品JSON |
| `is_draft` | `draft` | 草稿标识 |

当前代码已显式使用 `@TableField` 对上述列逐项映射，按新项目标准库结构执行。
